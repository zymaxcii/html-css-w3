Library, Bookmarks and Templates for re-use

As part of my learning culture, I like to read, learn, try and experiment with various things, including software and hardware. Sometimes I do feel overwhelmed by it and suffer from information overload. I don't need:-

All these stuff because I live a simple, minimalistic life style. programming skills. Web server. Wiki, etc.

    What I should try to achieve is just to build up a library of knowledges and resources that I can dip into, re-use it, when the need arises.

  • A library of codes in PG usb stick
  • A library of urls in Firefox bookmarks: Install, Repo, Tool
  • A library of ebooks in Ultra64GB usb stick.
With all these resources, I can further dip into any topic with ease! Proper collection and classification of information can solve the problem of information overload.
